# Liminal

A personal media library for the web.

Liminal is a set of tools which help people collect, enjoy, and archive their media and cultural interests, online and off. 

As a tool intended for daily use, Liminal should be accessable from:

- [ ] Web
- [ ] Chrome Extension
- [ ] Safari Extension
- [ ] Firefox Extension
- [ ] IE & Edge Extension
- [ ] Opera Extension
- [ ] Brave Extension
- [ ] App
- [ ] iOS
- [ ] Android
- [ ] Windows Local Desktop
- [ ] macOS Local Desktop
- [ ] Linux Local Desktop
